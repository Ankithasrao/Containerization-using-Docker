# Containerization-of-the-application-using-Docker 

### This repo demonstrates how to build the images using multi-stage dockerfile for Java and Nodejs applications

## Project Structure for Java-project
```
java-app/
│── src/
│   └── main/
│       └── java/
│           └── com/example/Hello.java
│── pom.xml
│── Dockerfile
│── .gitignore
│── README.md

```

## Project Structure for Nodejs-project
```
node-docker-app/
│
├── Dockerfile
├── package.json
├── server.js
└── README.md
```
